# 10xwork-find

### Głowna funkcjonalność
Zautomatyzowany proces szukania pracy w it w dziedzinie deweloperów java wysyłania i genrowania CV bazujac na wejściowym doświadczeniu i skilach


### Najmniejszy zestam funkcjonalności
- przyjmowanie skili skopiowanych z ogłoszenia pracy w postaci textu. Kopkuj wklej
- dodawanie i edycja bazowych skilli kandyata, np git, docker, java itp
- dopasowanie skili bazowego kandydata do wklejonych z pierwszego punktu i generowanie z bazowych
- skille z obu grup nie muszą się zgadzac na zasadzie git = git. Tylko np "znajomość systemu kontroli wersji VCS" =  git


### Co nie wchodzi w akres MVP
- brak generowani innych sekcji CV  jak edukacjia, szkolenia, kursy
- Importy innych plików  PDF itp
- system autoryzacji
- narazie bedzie tylko jeden użytkownik, ja
- zadnych aplikacji mobilnych
